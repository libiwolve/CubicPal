# 🐕 CubicPal

**一只住在你桌面上的方块小狗。**

> 目前处于早期开发阶段，我们正在构建第一个可互动的原型。

---

## ✨ 项目愿景

让操作系统不再是冰冷的图标，而是一个有生命的小世界。**CubicPal** 是一个可养成、可互动的桌面伙伴，它能感知你的文件，理解你赋予它们的意义，并通过邮件与你对话、成长。

第一个角色是一只由简单方块构成的小狗——它好奇、笨拙，却努力想弄懂你桌面上的“地球”、“月球”和“回收站”是什么。

---

## 🧠 核心理念

- **桌面即世界**：每个文件/文件夹都可以被赋予属性（标签、行为描述、情感基调），成为小狗世界的“景观”。
- **AI为脑，代码为身**：大语言模型负责理解和生成角色的语言与计划，Unity代码负责执行动作与交互。AI不直接操作角色。
- **邮箱作为交互界面**：所有指令、请求和故事都通过拟物化的“邮件系统”完成，避免开放的指令输入，保证成功率。
- **低美术成本，高情感表达**：方块狗狗通过耳朵、尾巴和像素眼睛的变化，用极简的动画传递丰富情绪。

---

## 🎯 第一期原型目标 (MVP)

- [ ] 在Unity中实现透明、置顶、可交互的桌面窗口
- [ ] 方块狗狗角色基础动画（视线跟随鼠标、摇尾巴、跳跃）
- [ ] 读取桌面路径并生成文件/文件夹列表
- [ ] 为物品赋予简单属性并保存
- [ ] 邮件系统原型（预设选项式交互）
- [ ] 接入大模型生成角色文本（故事、请求）
- [ ] 完整的“感知→联结→协助”三阶段养成流程演示

## 🐶 关于“方块小狗”

我们刻意选择了一个造型极简的角色作为起点。这让我们可以把精力集中在AI交互逻辑和桌面世界观的构建上，而不是复杂的骨骼动画。这种限制反而催生了独特的表达方式——用几何体的“歪头”、“抖动”和像素表情来传递情感。

后续会基于同一个底层框架，制作不同性格的角色模板（猫系、毒舌、热血导师等）。
